<h2 id="features">✨ Features</h2>

This digital business card is designed to be a highly interactive, modern web asset:
- **Fluid 3D Physics:** The card intelligently tracks mouse or touch movements, physically tilting in 3D space using spring-based physics for zero-latency responsiveness.
- **Interactive Flip Mechanics:** Clicking the card seamlessly flips it 180 degrees to reveal secondary information, maximizing screen real-estate.
- **Dynamic Theming:** Built-in Light (Rose) and Dark (Sky Blue) modes that completely swap the glassmorphism color palettes, text rendering, and background ambient glowing orbs.
- **Actionable VCard Generation:** Direct, client-side generation of a `.vcf` file allowing instant contact saving to address books.

<h2 id="tech-stack">� Tech Stack</h2>

This project leverages modern frontend technologies to deliver a buttery-smooth, interactive experience:

- **⚛️ Frontend Framework:** React 19 via Vite for lightning-fast module replacement.
- **🎨 Styling:** Tailwind CSS V4 combined with generic CSS properties for native, hardware-accelerated glassmorphism.
- **🎬 Animations:** Framer Motion (specifically utilizing `useSpring`, `useTransform`, and `useMotionValue` to manage the complex math for 3D rotation).
- **🔣 Utilities:** `lucide-react` / `react-icons` for scalable vector iconography and `qrcode.react` for dynamic standard contact rendering.
